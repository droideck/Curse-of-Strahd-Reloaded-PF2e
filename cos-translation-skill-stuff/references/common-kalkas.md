# Common калькi (English-to-Russian literal translations) and their fixes

A running catalogue. **Append every time a new calque is caught in editing.** Future-you needs the same list.

Each entry: English source → bad Russian (calque) → natural Russian.

## Idiomatic калькi (phrases)

| English | Calque (avoid) | Natural Russian |
|---|---|---|
| be subjected to X | подвергнуться X | напороться на X / попасть под X |
| within the framework of | в рамках | по; при; (often droppable) |
| achieve [effect] | достигнуть/достичь [эффекта] | вызвать, добиться, создать |
| have an impact on | оказать влияние на | повлиять на, задеть |
| make a decision | сделать решение | принять решение, решить |
| make an effort | сделать усилие | приложить усилие, постараться |
| there is/are | присутствует | есть, стоит, лежит, ходит |
| in addition | в дополнение | кроме того, к тому же, ещё |
| something happens to someone | что-то происходит с кем-то | что-то случается с кем-то |
| become more X | стать более X | (often) X-ее, обернуться X-им |
| antisocial behavior | антисоциальное поведение | недоверие, враждебность, отчуждение |
| emotional involvement | эмоциональная вовлечённость | привязанность, участие, близость |
| grow towards X | расти в сторону X | двигаться к X, становиться X |
| feel free to | чувствуйте себя свободно | не стесняйтесь, можете |
| at the end of the day | в конце дня | в итоге, в конечном счёте |
| make sense | иметь смысл | (often) понятно, разумно, ложиться |
| make sure | убедиться | проследить, проверить (often) |
| run into a problem | столкнуться с проблемой | напороться, столкнуться, увязнуть |
| keep in mind | держать в уме | помнить, не забывать |
| sounds good | звучит хорошо | хорошо, годится, пойдёт |

## Syntactic калькi (sentence-shape)

| English shape | Calque | Natural Russian |
|---|---|---|
| "X is Y" copula | X является Y | X — Y *(dash, no copula)* |
| "given that X, Y" | учитывая, что X, Y | раз X — Y; коли X — Y |
| "X being Y" | X являясь Y | (recast: X, который Y / X-Y / X. Он Y.) |
| English passive | X было сделано (passive) | Сделали X *(active)* |
| "having done X, Y" | сделав X, Y *(overused gerund)* | (sometimes natural, but watch overuse; break into two sentences) |
| English nominal style | X-ность чего-то | (concrete noun + verb) |
| English oxford comma list | А, Б, и В | А, Б и В *(no comma before и)* |
| "It is X that Y" cleft | Именно X, что Y | (restructure — Russian doesn't cleft like English) |
| "to do X, you need Y" | чтобы сделать X, вам нужно Y | (often natural; but watch second-person stiffness) |

## Punctuation калькi

| English habit | Calque | Russian |
|---|---|---|
| em-dash | X-Y or X--Y or X-Y | X — Y *(em-dash with spaces)* |
| outer quotes | "X" or 'X' | «X» |
| inner quotes | «"X"» | «"X"» (Russian uses «» outer, "" inner) |
| period inside quote | "X." | «X». *(period outside)* |
| comma inside quote | "X," | «X», *(comma outside)* |
| Oxford comma | A, B, and C | A, B и C *(no comma before и)* |

## Register-shift калькi (gaming / RPG specifics)

| English | Calque | Natural Russian |
|---|---|---|
| roll | роллить | бросить (кубик), кинуть |
| crit | критануть | выпасть критом, попасть критически |
| stat | стат | характеристика, показатель |
| buff / debuff | бафф / дебафф | усиление / ослабление *(or keep English in mechanical context)* |
| TPK | вайп | полное поражение, разгром, татаргат *(or keep TPK in mechanical context)* |
| level up | левелапнуться | подняться в уровне, получить уровень |
| roleplay | роулплеить | отыгрывать |
| GM screen | экран ГМа | ширма мастера |

**Note:** Some gaming terms (NPC, GM, party, V, XP, Recovery, Milestone XP) **stay English / transliterated** because that's table convention. The list above is for words where a natural Russian equivalent exists and the English form is colloquial slang, not technical.

These have shown up in the project text and been fixed. Watch for recurrences.

- **«лимиты»** → «пределы». Caught 2026-05-20 in player handout.
- **«антисоциального поведения»** → «недоверия и отчуждения». Caught 2026-05-20 in campaign contract.

## When in doubt

If you're unsure whether a phrase is a calque:
1. Read it aloud mentally in Russian.
2. Ask: would a native Russian writer in a gothic register write this?
3. If the answer is "maybe, in a translation" — it's a calque. Rewrite.
4. If the answer is "I've read this in Lermontov / Pushkin / Dostoyevsky" — it's fine.
